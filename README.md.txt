# Golden Ratio Flavor Structure from Modular Symmetry

This repository contains code to reproduce results from the paper "Golden Ratio Flavor Structure from Modular Symmetry at τ = e^{2πi/5}"

## Quick Start
1. Clone: `git clone https://github.com/YOUR-USERNAME/GoldenRatioModularFlavor.git`
2. Install: `pip install numpy scipy matplotlib`
3. Run: `python verify_results.py`

## Expected Results
- χ² = 18.3 for quark sector fit
- tanθ₁₂ = 0.641 (golden ratio prediction: 1/φ = 0.618)
- All predictions match Tables 1-2 in paper

## Files
- `verify_results.py` - Main verification script
- `model.py` - Core model implementation
- `requirements.txt` - Python dependencies
